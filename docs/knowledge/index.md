# Discovery Knowledge Base

Diese Knowledge Base enthält das Wissen aus dem "Product Discovery Handbuch" (Becker, Blaase, Milz).

## Struktur

- `mandat/` - Wissen für Arthur (Briefing, Auftragsklärung)
- `bedarf/` - Wissen für Finn (User Research, JTBD)
- `ideen/` - Wissen für Ida (Ideation, HMW)
- `test/` - Wissen für Theo (Experimente, Test Cards)

### Truth Curve
